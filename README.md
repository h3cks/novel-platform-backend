# Novel Platform — Backend

Backend частина вебплатформи для створення, публікації та читання новел.

## Стек
- Node.js / Express
- PostgreSQL
- Prisma (ORM)
- JWT auth
- Email (SMTP)

## Запуск локально
1. Перейти в папку backend:
   ```bash
   cd backend
Встановити залежності:
npm install

Створити <b>.env</b> на основі <b>.env.example</b>

Запуск:
```bash
npm run dev
